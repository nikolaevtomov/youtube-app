## Simple React App with ES6

## Includes

* Gulp
* Bower
* Browserify
* Reactify - Help to transform JSX (consider to use babelify later)
* Watchify support! (Sourcemap also!)
* livereload (BrowserSync)

## Getting Started

```
Run: $ npm install
```
```
Run: $ gulp watch
```
```
Run: $ npm test
```
```
Run: $ gulp build
```
